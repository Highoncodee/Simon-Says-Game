🎮 Simon Says Game

An interactive Simon Says memory game built using HTML5, CSS3, and JavaScript. The game challenges players to memorize and repeat an increasingly long sequence of colors while demonstrating core JavaScript concepts like DOM manipulation, event handling, arrays, and game logic.

✨ Features

- Interactive Simon Says gameplay
- Animated button flash effects
- Increasing difficulty with each level
- Random color sequence generation
- High score tracking
- Game over detection and restart
- Smooth user interactions using JavaScript
- Clean user interface

🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
